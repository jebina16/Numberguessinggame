# Number Guessing Game

A simple Java console application where the user tries to guess a randomly generated number between 1 and 100 within 5 attempts.

## Features

* Random number generation
* User input using Scanner
* Hint messages (higher/lower)
* Limited attempts
* Simple console interface

## Technologies Used

* Java
* Scanner Class
* Math.random()

## How to Run

1. Compile:
   javac GuessGame.java

2. Run:
   java GuessGame
